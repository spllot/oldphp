InTimeGo即時服務

InTimeGo即時服務資訊網 (或稱 InTimeGo即購網) 是全民性的行動商務平台，係針對一般傳統商店、行動商店 (攤販、行動販售車、補貨物流車...等)、運輸&共乘、即時人力、即時活動以及個人商務，進行服務搜尋以及商品販售的網路平台，對普羅大眾而言，這是一個能拉近供需距離與創造服務價值的網站，可以為買賣雙方創造更為便利與快速的商業關係。

另一方面也藉由推展「全民皆商」與「全民行銷」的構想，冀望創造更多元的商務與獲利的機會，平台運作把販售商品的部分盈餘分享給全民，只要能為平台做出貢獻的全民都能受到回饋；平台的理想願景是，希望未來實現「企業員工全民化」目標，本站實施方式以「傳播式商品」作為敲門磚，期能引導各行各業以相關模式跟進，務使全民不再有失業的困擾!

本站將經由以下兩點, 達到利潤分享於買賣雙方:

(1). 提供低門檻費用之團購商品販售服務，讓賣家能保有販售商品的利潤，買家能享有高品質的消費服務。另提供賣家建置非金流商品頁面 (新品、即期品與中古貨，皆可設定為非金流商品)，不需抽取任何販售服務費用，讓賣家享有0成本之商品行銷販售機會。

(2). 網站會員可以享有部落格行銷與商品傳播獲利之機會，本站將商品行銷經費回饋給全民，凡對站內團購商品有介紹、購買或資訊傳播行為，皆可經由紅利回饋方式獲得儲值金，因而享有0元購買商品以及賺取現金之機會。

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

即購網_行動APP使用與功能說明 

前台服務陳列與搜尋功能

使用者下載 InTimeGo APP之後 (下載方式, 請參考 InTimeGo APP下載說明)，點選進入APP首先出現GPS搜尋中畫面 (如圖一)，進行抓取使用者的手機方位，做為本地服務搜尋商品遠近陳列之參考點 ; 當系統得到使用者位置後，畫面將導入「本地服務」功能之列表 (如圖二, 此頁面可點選之功能尚有「宅配服務」、「本地團購」以及「宅配團購」)，使用者可以依據商家提供之服務&商品遠近，決定是否符合於自己的需求而做呼求。使用者若需更有效率搜尋本地之服務&商品，也可以點選上方的「選項」按鍵，進入到「類型」&「分類」搜尋服務選項 (如圖三)，點選此搜尋服務選項功能，可以幫助使用者更精簡收斂與更快速搜尋前台想要尋找的服務與商品。

補充說明: 當任何頁面發生無法讀取資料之狀況時，且其他功能鍵都無法反應之狀態，使用者可將裝置顯示畫面傾斜90度後，可以讓頁面重新抓取手機方位，並且回到首頁。

服務說明與商家其他服務查詢

使用者可以點選前台各列表的服務或商品圖示，進入商家服務說明頁面瀏覽服務內容，此頁面可以將商家服務或商品分享至Facebook，或將此商家加入收藏，以便追蹤商家行蹤 (參考圖四)，為更詳盡了解服務內容與商家資訊，可以點選「服務&資訊說明」以及「業者(商家)資訊說明」按鍵(參考圖五) ; 若為「本地服務」之項目，圖五展示尚有「商家遞送路徑」「買家前往路徑」以及商家「目前服務位置圖」功能，讓消費者更便利於買賣交易進行 ; 另外，消費者也可以點選「金流交易評價」，以了解該商家於金流商品服務評價積分之高低，做為購買商家服務或商品前之參考 ; 消費者也可以經由「詢問問題」「發表評論」按鍵，直接對商家諮詢問題或發表對此服務之評論。

如果我們已選擇購買商家的服務或商品，但需求商家是否有其他服務，可以一併考量購買時，可在前台商品列表點選「商家其他服務」，進入商家專屬頁面，選購該商家他服務或商品 (如圖六所示)，欲了解商家專屬頁面之商品或服務集合，此專屬頁面仍有區分「本地服務」、「宅配服務」、「本地團購」以及「宅配團購」可以點選，如欲進一步了解商家資訊，尚可在頁面最上方處點選商家名稱，閱讀商家介紹頁面。

會員登入功能_買家身分登入

在APP 上方按鍵「登入」功能，可讓使用者註冊會員以及登入會員 (如圖七)，如果會員點選身分登入後，畫面將導向前台商品列表，同時產生黃色小框顯示會員名稱，會員可由APP 上方之「選項」，按下「進入用戶中心」進入會員服務後台 (如圖八)，如果會員以買家身分登入，進入服務後台顯示功能如圖九所示，可以經由顯示之「訂單交易查詢」按鍵，查詢過往訂單以及目前訂單之狀態，經由「我的服務收藏」按鍵，追蹤商家目前服務地點。

會員登入功能_商家身分登入

當以商家身分登入用戶中心首頁，頁面會有三個主要按鍵呈現，分別為「我的位置與狀態設定」、「本地服務交易記錄」以及「本地服務處理資訊」，但如果商家並未申請金流商家功能時，此用戶中心按鍵則僅呈現「我的位置與狀態設定」功能 (如圖十所示) ; 當商家申請為金流商家身分時，用戶中心首頁下方之「購買通報設定方式」設置則會產生作用，「購買通報設定方式」顧名思義為訂單發生時通報商家知悉之方式，商家可以依據通報即時處理訂單交易 ; 接著頁面下方「撮和與即時遞送商品」係設定商家是否提供撮和與即時遞送服務，若商家提供此服務可滿足商品遞送方圓距離的狀況下，就會產生商品撮和之機制，此撮和之機制會等待商家決定是否接受買家的遞送訂單，如果買家與商家之遞送方圓距離超過設定，則商品撮和與遞送機制將不會發生。

當點選進入「我的位置與狀態設定」，顯示如圖十一與圖十二頁面，共有四大項目分別為「本地服務營業狀態設定」、「我的行動服務位置來源設定」、「我的行動服務位置」與「商家即時遞送商品服務」; 以上各項目皆有文字陳述功能，在此不做細節贅述，僅針對部分重要功能做補述，例如，用戶中心首頁下方之「撮和與即時遞送商品」功能點選[開啟]，相當於「我的位置與狀態設定」頁面自動點選三項功能設定，以符合「撮和與即時遞送商品」功能運行，第一項設定為「本地服務營業狀態設定」自動點選第三項，第二項設定為「我的行動服務位置來源設定」自動點選第二項，以及第三項設定為「商家即時遞送商品服務」自動點選[設置]，以下段落為此三項設定之功能說明:

「本地服務營業狀態設定」點選第三項: 功能係為以商家身分登入時，其商品或服務則相應上架，登出時相應下架，此設定方式適合於進行行動服務之商家。在此另外說明運輸服務之設定，除需點選此設定之外，還需在[計程車服務狀態]項目做點選，以使搭乘者能夠了解業者的運輸狀態，如考量進行共乘服務時，則需點選[載客中(共乘)行駛目標]，並在下方點選並輸入行駛目標住址 (只要業者徵詢乘客開放共乘狀況下, 可以在此輸入行駛目標住址)，以使欲共乘者知道車輛之行駛目標而加入服務。

「我的行動服務位置來源設定」 點選第二項: 係以行動裝置APP連結本站時，行動服務位置來自行動裝置所發送，商家所在方位將由行動裝置自動上傳到主機 (上傳條件: 30秒位移超過10米)，以反應商家商品或服務實際方位，便利於買家搜尋所需之服務與方位。

「商家即時遞送商品服務」點選[設置]: 前台服務圖片上即會貼上「即時遞送」標籤，告知此商品有支援即時遞送到府服務。
